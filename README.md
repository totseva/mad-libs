# mad-libs
🎮 Mad Libs Story Game

A small interactive web game where you create a short story by filling in a few words.

✨ How it works
- Enter a character name
- Add an adjective
- Add a mysterious word
- Get a short story generated from your inputs

The game supports both English and Russian, with slightly different storytelling styles for each language.

🌙 Features
- Dynamic prompts and story generation
- Language switch (EN / RU)
- Playful UI with styled text highlights
- “Start over” functionality

🛠 Built with
- HTML
- CSS
- JavaScript (vanilla)

🎯 Goal

A simple project to practice DOM manipulation, user input handling, and creating a fun interactive experience.

Try different words and see how strange the story can become ✨
